# Online Voting System

A simple web-based voting system built using **HTML, CSS, and JavaScript**.
This project demonstrates how users can select a candidate and cast a vote through a simple user interface.

## Features

* Simple and clean voting interface
* Candidate selection using buttons
* Instant vote confirmation message
* Beginner-friendly frontend implementation

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

online-voting-system
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run the Project

1. Download or clone the repository
2. Open `index.html` in your browser
3. Select a candidate to cast your vote

## Future Improvements

* Add login and authentication
* Store votes in a database
